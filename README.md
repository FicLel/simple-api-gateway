# API GATEWAY

This is a simple API GATEWAY doing a proxy implementation for micro
services, can be improved

## Enviroment Variables

- `DB_HOST` 
- `DB_PORT` 
- `DB_USER` 
- `DB_PASSWORD` 
- `DB_DATABASE` 
- `SALT_TOKEN`

## Database fields

Using MariaDB we just need a users table with the next fields

- email: Varchar
- password: Varchar
- id: Int
